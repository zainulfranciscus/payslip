These are assumptions made about this assignment:
  • Assumption 1: The application will produce 0 as income tax when Employee Salary is not between maximum and minimum income in a table.
  • Assumption 2: The application will produce 0 as income tax when Employee's pay period is before the tax table's start date.
  • Assumption 3: The application will produce a payslip with an income tax when the salary is between the min and maximum income tax.
  • Assumption 4: Tax and salary can be described in decimals using dot, e.g: 1800.00
  • Assumption 5: When a tax is described in non-numerical values, then that tax is treated as 0.
  • Assumption 6: When salary is describes in non-numerical values, then that salary is treated as 0.
  • Assumption 7: When tax table dates are invalid, e.g: is not a number, or a number that won't be inside a calendar, then that tax will not be used to calculate a payslip.
  • Assumption 8: -1 in tax table maximum income column means that, the tax should be applicable to employee where his income does not match other tax's maximum income in the same table.
  • Assumption 9: When employee payment period is not in the same month, then there will be no payslip produced for that employee.
  • Assumption 10:0 payslip will be produced when tax table is empty
  • Assumption 11:0 payslip will be produced when no employee information is provided
